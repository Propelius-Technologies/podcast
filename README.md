
## Requirements
### Yarn installation

This project will need yarn to run

```bash
$ npm install -g yarn
```
### Install
```bash
$ git clone git@github.com:Propelius-Technologies/podcast.git
$ cd PROJECT_TITLE
$ yarn install
```

### Running the project
```bash
$ yarn start
```
### Simple build for dev
```bash
$ yarn build
```
### Simple build for production
```bash
$ yarn build
```
### Versions
- Yarn version : 1.22.18
